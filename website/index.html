Punctuation & Ellipses Differences (Robust)

Matching Robustness

Our keying engine is built to throw away all punctuation marks. Because of this, sentences that differ only by spacing or punctuation (like a period . vs. an ellipsis ...) naturally resolve to the exact same key.

How to Fix Phrasing Mismatches

When the English dialogue source in the JSON deviates from the game's actual English text, the plugin will fail to match.

Remedy Options:

  1. Update the text_en field in the JSON file to exactly match the game's wording.
  2. Or specify an explicit key in the JSON that matches the game's normalized wording.
